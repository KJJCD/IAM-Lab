# IAM-Lab
This is a lab where I will creating an on prem Windows environment, using Entra Connect sync to make that environment Hybrid, then integrating SSO into Entra with Okta

## Overview
Hands-on IAM lab simulating enterprise identity architecture using Active Directory, Okta, and Microsoft Entra ID.

## Goals
- Build on-prem AD as source of truth
- Implement RBAC using groups
- Configure SSO (SAML/OIDC) with Okta
- Automate provisioning (SCIM)
- Implement JML (Joiner/Mover/Leaver)
- Configure Conditional Access & MFA
- Perform Access Reviews
- Automate identity tasks with PowerShell

## Tech Stack
- Windows Server 2019 (AD DS, DNS)
- Microsoft Entra ID
- Okta
- Slack / Google Workspace
- PowerShell
- VirtualBox

## Architecture (Phase 1)
On-Prem AD (DC01) → Identity Source
